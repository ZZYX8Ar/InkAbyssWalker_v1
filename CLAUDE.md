# InkAbyssWalker - Unity 游戏项目

## 项目概述

InkAbyssWalker 是一个 Unity 游戏项目，使用通用渲染管线（URP）。

## 项目结构

- `Assets/Scripts/` — 游戏脚本
  - `BaseFrameWork/` — 基础框架
  - `PlayerController.cs` — 玩家控制器
  - `State/` — 状态管理
  - `Item/` — 物品系统
  - `ExcelData/` — Excel 数据配置
- `Assets/ArtRes/` — 美术资源
  - `Animation/` — 动画资源
- `Assets/Scenes/` — 场景文件
- `Assets/Settings/` — 项目设置（URP 等）
- `Assets/Editor/` — 编辑器扩展
- `Assets/Resources/` — 动态加载资源

## 技术栈

- Unity + C#
- Universal Render Pipeline (URP)
- Excel 数据驱动

## 开发规范

- 遵循现有命名规范和代码风格
- 新功能优先在现有框架基础上扩展
